## Projects

### [aersia-vip-player-self-hosted-fork](https://github.com/AC-DAC/aersia-vip-player-self-hosted-fork)
Self-hosted music player running on a Raspberry Pi 4B. Demonstrates end-to-end Linux server administration: Nginx, Let's Encrypt TLS (DNS-01 challenge via Cloudflare), dynamic DNS automation via Cloudflare API, PWA support, and cron-based maintenance. Forked and extended with sequential playback mode and full localStorage persistence.

`Nginx` `Let's Encrypt` `Cloudflare` `Bash` `Linux` `PWA`

---

### [FitForge](https://github.com/AC-DAC/FitForge)
Privacy-first workout tracking app for Android. No ads, no account required, all data stored on-device. Features custom workout creation, guided session mode, stats and insights, reminders, and QR code workout sharing. Built in React Native / Expo with a custom Material Design-inspired design system. Currently in closed testing on the Google Play Store. CI/CD pipeline via GitHub Actions → EAS Build → Play Store internal track. Jest unit test suite. Lefthook + gitleaks for pre-commit secret scanning.

`React Native` `Expo` `GitHub Actions` `EAS Build` `Jest` `Android`
